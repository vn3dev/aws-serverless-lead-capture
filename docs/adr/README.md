# Architecture Decision Records (ADR)

This directory contains formal architectural decisions made during the design of this project.

Each ADR documents:
- Context
- Decision
- Alternatives considered
- Consequences and trade-offs

The goal is to provide transparency and rationale behind key architectural choices.
